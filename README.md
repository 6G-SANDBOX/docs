# 6G-SANDBOX TOOLS DOCS

## Getting Started Locally

Clone repository

```bash
git clone https://github.com/6G-SANDBOX/docs.git
```

Install [bun](https://bun.sh/)

```bash
curl -fsSL https://bun.sh/install | bash
```

Access to docs repository

```bash
cd <docsPath>
```

Install libraries

```bash
bun install
```

Start server

```bash
bun start
```
