# 6G-Sandbox Experimenters

An Experimenter in 6G-Sandbox is whomever who needs to use Trial Networks (TN) for testing or research purposes.
Trial Networks are isolated environments that allow users to experiment with various components and configurations without affecting the main network.

Unless a site's policies state otherwise, Experimenters are also responsible for deploying and managing their own Trial Networks.
